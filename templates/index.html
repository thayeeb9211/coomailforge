System Details  GET /api/v4/systems/{id}
Enter a System ID and click Query.
System Summary  GET /api/v4/systems/{id}/summary
Lifetime energy & current power output.
Devices / Microinverters  GET /api/v4/systems/{id}/devices
Microinverter serial numbers and device info.
Energy Lifetime  GET /api/v4/systems/{id}/energy_lifetime
Daily production history since system install.
Production Telemetry  GET /api/v4/systems/{id}/telemetry/production_micro
Solar production telemetry per granularity.
Consumption Telemetry  GET /api/v4/systems/{id}/telemetry/consumption_micro
Site consumption telemetry (requires consumption meter).
Inverters Summary  GET /api/v4/systems/{id}/inverters_summary_by_envoy_or_site
Per-inverter performance summary grouped by Envoy.
